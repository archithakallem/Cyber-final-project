What is CyberScan?

A comprehensive security analysis tool combining a FastAPI backend with a Streamlit frontend.

  • Scans target systems & networks
  • Enriches data via threat intelligence
  • Calculates aggregated risk scores
  • Stores scan history & sends actionable alerts

Key Features

  • Dual Architecture: Scalable FastAPI API + Streamlit UI
  • Discovery: Nmap-based open-port detection
  • Intelligence: Deep VirusTotal integrations
  • Scoring Matrix: Exposure, Threat, Context & Overall Risk

Press ⬇️ for more features

Advanced Features

  • Rich Dashboard: Summary, Analysis, Visuals, Risk Maps
  • Data Tracking: SQLite history logs for auditing
  • Proactive Alerts: Email notifications for high risk scenarios (Score > 70)

How It Works

  1. User provides Target & VT API Key
  2. UI triggers the /scan/{target} endpoint
  3. Backend runs Nmap for exposed ports
  4. Queries VirusTotal for deep threat Intel
  5. Normalizes data & calculates Risk Scores
  6. Saves to DB & Triggers Email Alerts

Supported Targets

  • 🌐 Domains (e.g., example.com)
  • 🖥️ IP Addresses (e.g., 8.8.8.8)
  • 🔗 Uniform Resource Locators (URLs)
  • 📄 File Hashes
